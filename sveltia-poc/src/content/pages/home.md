# Welcome to Our Proof of Concept

This is a sample homepage demonstrating Sveltia CMS integration with Astro.

## Key Features Being Tested

- **Git-based Workflow**: All content commits to Git repository
- **Visual Editing**: Non-technical users can edit via `/admin` interface
- **Version Control**: Every change tracked in Git history
- **Type Safety**: Astro Content Collections ensure schema validation

## Test Checklist

1. Access `/admin` interface
2. Edit this page content
3. Save and commit changes
4. Verify Git commit created
5. Test rollback functionality
6. Upload images via CMS
7. Create new blog post
8. Test on mobile device

This POC validates Sveltia CMS viability before committing to full monorepo architecture.
