# Website Site Architecture
**Harbour View General Practice**
**Prepared:** November 2025

## Executive Summary

This site architecture document defines the complete information architecture, URL structure, navigation hierarchy, and internal linking strategy for Harbour View General Practice. The architecture prioritizes patient user experience, SEO performance, and clear pathways to appointment booking while establishing authority in the Northern Beaches general practice market.

### Architecture Overview
- **Total Pages:** 28 pages
- **Primary Navigation Items:** 6
- **Service Pages:** 12
- **Supporting Content Pages:** 10
- **Functional Pages:** 6
- **URL Structure:** Clean, keyword-optimized, logical hierarchy
- **Mobile-First Design:** All navigation patterns optimized for mobile users

## 4. URL Structure Guidelines

### 4.1 URL Best Practices

**Format:**
```
https://harbourviewgeneralpractice.com.au/[section]/[page-name]/
```

**Rules:**
- All lowercase
- Hyphens for word separation (not underscores)
- No special characters or numbers
- Keyword-rich but concise
- Logical hierarchy
- Trailing slash for consistency
- No date-based URLs (except blog posts if desired)

**Examples:**
- `/services/chronic-disease-management/` ✓
- `/Services/ChronicDiseaseManagement/` ✗
- `/services/cdm/` ✗ (unclear abbreviation)
- `/page-id-42/` ✗ (meaningless)

---

### 4.2 URL Hierarchy

**Level 1 (Section):**
- `/about/`
- `/services/`
- `/patients/`
- `/health-hub/`
- `/contact/`

**Level 2 (Page):**
- `/services/womens-health/`
- `/patients/new-patients/`
- `/about/our-doctors/`

**Level 3 (Sub-page if needed):**
- Generally avoided for simplicity
- Exception: `/health-hub/category/post-name/` (blog categories)

## 6. SEO and Technical Considerations

### 6.1 Page Metadata

**Every Page Includes:**
- Unique, descriptive H1 (only one per page)
- Title tag (50-60 characters, includes primary keyword + location + practice name)
- Meta description (150-160 characters, compelling summary with CTA)
- Schema markup (LocalBusiness, MedicalBusiness, Physician)

**Example (Women's Health Page):**
```
H1: Comprehensive Women's Health Services
Title: Women's Health Services | Harbour View General Practice, Northern Beaches
Meta Description: Expert women's health care including cervical screening, contraception, menopause and pregnancy support. Book with our experienced GPs on the Northern Beaches.
```

---

### 6.2 Image Optimization

**All Images Include:**
- Descriptive file names (womens-health-consultation.jpg, not IMG_1234.jpg)
- Alt text (accessibility and SEO)
- Compressed file size (max 200KB for web)
- Appropriate dimensions (no oversized images)

**Example:**
```
File: northern-beaches-gp-consultation.jpg
Alt: "Female GP consulting with patient at Harbour View General Practice"
```

---

### 6.3 Mobile Responsiveness

**Requirements:**
- Mobile-first design approach
- Touch-friendly navigation (min 48px tap targets)
- Fast loading (<3 seconds on 4G)
- Readable text without zooming (min 16px font)
- Click-to-call phone numbers
- One-tap appointment booking
- Simplified forms for mobile

---

### 6.4 Page Load Performance

**Optimization Targets:**
- Page load time: <2 seconds
- Largest Contentful Paint (LCP): <2.5s
- First Input Delay (FID): <100ms
- Cumulative Layout Shift (CLS): <0.1
- Lazy loading for images below fold
- Minified CSS/JS
- Browser caching enabled

---
