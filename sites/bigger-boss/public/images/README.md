## Image Optimization Checklist

Before uploading images to this directory:

- [ ] Compress images (target: <200KB for large images, <50KB for avatars)
- [ ] Convert to WebP format where possible (with JPEG/PNG fallbacks)
- [ ] Ensure correct dimensions match specifications above
- [ ] Verify color accuracy (Deep Navy #1a202c, Electric Blue #3B82F6)
- [ ] Add descriptive file names (no spaces, use hyphens)
