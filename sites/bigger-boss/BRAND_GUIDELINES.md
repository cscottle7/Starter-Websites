## Design Archetype: Professional B2B SaaS

**Rationale:**

Bigger Boss is an AI-powered marketing strategy service that delivers comprehensive marketing strategies in 90 minutes instead of 8-12 weeks. The service targets Australian agencies and professional services with AHPRA/ASIC compliance built-in, 85+ quality score guarantees, and up to 85% cost savings.

**Why Professional B2B SaaS Archetype:**
- **Target Market:** Australian professional services requiring compliance (AHPRA, ASIC, Law Society)
- **Decision Makers:** Agency owners, marketing managers, CFOs evaluating ROI
- **Competitive Landscape:** Competing against traditional agencies ($15K-$35K pricing)
- **Trust Requirements:** Compliance, quality guarantees, professional credibility essential
- **Brand Keywords:** Professional, reliable, compliant, data-driven, results-focused

**Archetype Characteristics:**
- Deep Navy + Vibrant Blue color palette (authority + innovation)
- Inter 400-700 typography (readable, professional, modern)
- Moderate border-radius (0.5-0.75rem) - approachable yet professional
- Subtle shadows (professional depth without flashiness)
- High contrast, accessible, data-driven visuals

## Typography

**Font Stack:**
- **Primary:** Inter (sans-serif)
  - Weights: 400 (Regular), 600 (Semi-Bold), 700 (Bold)
  - Source: Google Fonts
  - Character: Modern, professional, highly readable

- **Accent (Optional):** IBM Plex Serif (serif)
  - Weights: 400 (Regular), 600 (Semi-Bold)
  - Usage: Editorial content, testimonials (adds warmth)

**Type Scale:**
- **H1:** 40px (2.5rem) - Hero headlines
- **H2:** 32px (2rem) - Section headings
- **H3:** 24px (1.5rem) - Subsection headings
- **H4:** 20px (1.25rem) - Card titles, feature headings
- **Body Large:** 18px (1.125rem) - Hero subheadlines, intro paragraphs
- **Body:** 16px (1rem) - Standard body text
- **Small:** 14px (0.875rem) - Captions, metadata, fine print

**Font Weights:**
- **Headlines (H1-H2):** 700 (Bold) - Clear hierarchy, professional
- **Subheadings (H3-H4):** 600 (Semi-Bold) - Readable distinction
- **Buttons/CTAs:** 600 (Semi-Bold) - Actionable clarity
- **Body Text:** 400 (Regular) - Comfortable reading

**Line Heights:**
- Headlines (H1-H2): 1.2 (tight but readable)
- Subheadings (H3-H4): 1.3-1.4 (comfortable)
- Body: 1.6 (optimal readability)
- Small: 1.5

## Visual Style

**Geometry:**
- **Button Radius:** 0.5rem (8px) - modern, approachable
- **Card Radius:** 0.75rem (12px) - soft, professional
- **Image Radius:** 0.5rem (8px) - consistent with buttons

**Shadows:**
- **Subtle (Default):** `0 1px 3px 0 rgb(0 0 0 / 0.1), 0 1px 2px -1px rgb(0 0 0 / 0.1)`
- **Card (Moderate):** `0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1)`
- **Hover State:** `0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.05)`

**Borders:**
- Width: 1px (minimal, clean)
- Color: `#E2E8F0` (neutral-200)
- Style: Solid, no decorative borders

**Spacing:**
- Section Padding: 5rem vertical (80px) - comfortable breathing room
- Component Padding: 2rem (32px)
- Element Gaps: 1rem (16px) - consistent rhythm

**Animations:**
- Duration: 0.2s (fast, responsive)
- Easing: `ease-in-out`
- Properties: transform, opacity, box-shadow, color

## Voice & Tone

**Brand Personality:**
- **Professional:** "AHPRA/ASIC compliant strategies"
- **Results-Driven:** "90 minutes vs 8 weeks delivery"
- **Data-Focused:** "85+ quality score guaranteed"
- **Cost-Conscious:** "Up to 85% cost savings"
- **Trustworthy:** "100% compliance record across 50+ projects"

**Writing Guidelines:**
- Use active voice ("Deliver strategies in 90 minutes" not "Strategies are delivered in 90 minutes")
- Lead with benefits, follow with features ("Same-day delivery" before "97 AI agents")
- Quantify everything ("85+ quality score" not "High quality")
- Australian English spelling (optimise, colour, analyse)
- Professional without corporate jargon
- Confident without hype (back claims with data)

## Content Guidelines

### SEO Best Practices

- All pages must have unique `<title>` tags (50-60 characters)
- All pages must have unique meta descriptions (150-160 characters)
- All images must have descriptive alt text
- Blog posts must include complete frontmatter (title, description, publishDate, author, image, imageAlt)

### Accessibility Requirements

- Maintain single H1 per page
- Use semantic HTML structure
- Ensure color contrast meets WCAG 2.1 AA standards (4.5:1 for normal text)
- All interactive elements must be keyboard accessible

### AI Optimization

- Use semantic HTML for content structure
- Include schema.org JSON-LD markup on key pages
- Write clear, descriptive headings for scannability
- Use descriptive anchor text for links
