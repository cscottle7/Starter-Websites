## Visual Design

### Color Palette

#### Primary Color
- **Hex:** #1a202c
- **Usage:** CTAs (buttons), links, key UI elements, primary headings
- **Accessibility Note:** Ensure 4.5:1 contrast ratio with white text

#### Secondary Color
- **Hex:** #2d3748
- **Usage:** Accents, secondary CTAs, borders, supporting UI elements

#### Accent Color (Optional)
- **Hex:** #f59e0b
- **Usage:** Highlights, hover states, icons (use sparingly for visual interest)

#### Neutral Colors
- **Background Light:** #ffffff (White), #f7fafc (Light Gray)
- **Background Dark:** #1a202c (Dark Gray), #2d3748 (Charcoal)
- **Text Primary:** #1a202c (Dark Gray)
- **Text Secondary:** #4a5568 (Medium Gray)
- **Borders:** #e2e8f0 (Light Gray)

#### Semantic Colors (Alerts, Status Messages)
- **Success:** #10b981 (Green)
- **Error:** #ef4444 (Red)
- **Warning:** #f59e0b (Amber)
- **Info:** #3b82f6 (Blue)

---

### Typography

#### Headings
- **Font Family:** Inter, sans-serif
- **Weights:** 700 (bold for all headings)
- **Usage:** All H1-H6 elements

#### Body Text
- **Font Family:** Open Sans, sans-serif
- **Weights:** 400 (regular), 600 (semi-bold for emphasis)
- **Usage:** Paragraphs, lists, general content

#### Code/Monospace (if applicable)
- **Font Family:** Fira Code, monospace
- **Usage:** Code snippets, technical documentation

---

### Spacing Philosophy

- **Generous whitespace:** Use `space-y-8` and `space-y-16` liberally between sections
- **Breathing room:** Avoid cramped layouts - minimum 24px (`space-y-6`) between content blocks
- **Section Padding:** `py-16` (desktop), `py-8` (mobile)
